# 🎬 AttentionX AI

AI-powered system that converts long videos into short, engaging clips using transcription, highlight detection, and smart cropping.

---

## 🚀 Features

- Speech-to-text using Whisper
- AI-based highlight detection using Gemini API
- Audio-based fallback detection
- Smart vertical cropping using MediaPipe
- Automatic clip generation
- Streamlit UI

---

## 🛠️ Tech Stack

Python, Streamlit, Whisper, MediaPipe, OpenCV, MoviePy, Gemini API

---

## 📦 How to Run

Install dependencies:
pip install -r requirements.txt

Run the app:
streamlit run app.py

---

## 🔐 API Setup (Optional)

setx GEMINI_API_KEY "your_api_key_here"

Restart terminal after setting.

---

## ⚠️ Performance Note

Use short videos (20–60 seconds) for best performance.

---

## 🎯 How It Works

1. Upload video
2. Transcribe audio
3. Detect highlights
4. Crop video
5. Generate clips
